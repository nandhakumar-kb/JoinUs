# JoinUs

A React + Vite donation website supporting Ko-fi and UPI payments.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

- **GitHub Pages:** Push to main branch (auto-deploys)
- **Vercel:** Import repository at vercel.com
